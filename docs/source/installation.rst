Installation
============

Quick Install
-------------

Install Weather Tools using pip:

.. code-block:: bash

   pip install weather-tools

Development Installation
------------------------

For development:

.. code-block:: bash

   git clone https://github.com/varma1221/doc-test-project.git
   cd doc-test-project
   pip install -e .

Dependencies
------------

- Python 3.8 or higher
- No external dependencies required!
